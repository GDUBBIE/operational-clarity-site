Privacy Policy

This page summarizes the data flow currently implemented in Operational Clarity as of March 24, 2026. Users can complete the diagnostic review anonymously unless they choose to submit contact details for next-step help.

What the product collects

  • Messages entered into the diagnostic review.
  • Assistant replies, extracted fields, findings, normalized business-type tags, recommendation IDs, and confidence fields.
  • Session metadata such as timestamps, stage progression, page URL, source/channel fields when available, and whether contact details were submitted.
  • If you choose to request next-step help, the current build captures name, email, company, package interest, phone if provided, message text, and your follow-up contact consent.

How review information is used

  • Review information is collected to provide diagnostic guidance, improve the quality of the review, and improve Operational Clarity over time.
  • If a user requests follow-up help, the submission may also be used to prepare for that conversation.
  • We do not sell personal information.

How consent works in the product

  • Before the first review message is sent, the user must agree to the Terms of Use, acknowledge the Privacy Policy, and accept that the review provides diagnostic guidance only.
  • Contact consent for follow-up is separate and is requested only if the user chooses to move into next-step help.
  • The current build does not bundle service follow-up consent with optional marketing consent.

What is stored in the browser

  • The active review session is stored in local browser storage so a session can be restored if the page reloads.
  • Anonymous assessment history, anonymous archive snapshots, and structured intelligence snapshots are also stored locally for benchmarking and restore behavior.
  • If no external lead endpoint is configured, contact-form submissions are stored locally in browser storage.

What is stored in Supabase

  • Anonymous or identified session records may be written to Supabase through the `session-intelligence-ingest` function.
  • Current tables store transcript/message history, extracted JSON, findings JSON, recommendation JSON, normalized business features, symptom tags, cause tags, recommendation links, confidence values, and product-learning fields.
  • If contact is submitted, the session record is updated to reflect `contact_submitted` status.

What is sent to AI providers

The `business-diagnostic-agent` function sends the review conversation to the configured OpenAI API endpoint together with the internal system prompt needed to produce the diagnostic response. That means any information typed into the review may be processed by the configured AI provider.

Do not submit passwords, bank account numbers, medical information, legal claims, employee personal data, or other highly sensitive confidential information.

Human review for next-step help

Operational Clarity provides AI-assisted diagnostic guidance. If a user requests next-step help, the review may also be assessed by a human advisor from EHI Studio, a division of EHI Group, as part of follow-up and advisory work.

Cookies and analytics

The current build does not include third-party web analytics scripts, advertising pixels, or cookie consent tooling. The implemented persistence mechanism in the browser is localStorage, not tracking cookies.

Retention and deletion

  • The current build does not implement an automatic deletion schedule for Supabase session records.
  • The active local session is cleared when the user resets the review or when the build version changes.
  • Local benchmark and archive stores are capped in code by item count, not by time: anonymous score history is capped to 100 records, anonymous assessment archive to 250 records, and structured intelligence archive to 300 records.
  • If no external lead endpoint is configured, locally stored leads remain in browser storage until manually cleared.

Diagnostic guidance only

Operational Clarity provides business diagnostic guidance only. It is not legal, tax, accounting, or financial advice.
